## 기능 상세

### 1. 단순화된 목소리 설정

**기존:**
- 6개 영어 목소리 (Rachel, Adam, Josh, Liam, Emily, Daniel)
- 안정성, 유사성 부스트, 스피커 부스트 설정

**개선:**
- 2개 디폴트 목소리 (남자1, 여자1)
- 성별 선택만으로 간단히
- 고급 설정은 숨김 (선택사항)

### 2. 보이스 클로닝 기능

**단계:**
1. 오디오 샘플링 (마이크로 녹음 또는 파일 업로드)
2. 샘플링 데이터 업로드 (10초 이상)
3. 목소리 생성 (약 5-10분 소요)
4. 생성된 목소리 테스트
5. 목소리 저장 및 사용

**기능:**
- 내 목소리 생성 (최대 3개)
- 목소리 이름 설정
- 목소리 삭제
- 목소리 테스트 재생

### 3. API 키 선택사항

**데모 모드:**
- API 없이도 기본 목소리 (남자1, 여자1) 사용 가능
- 보이스 클로닝만 API 필요

**운영 모드:**
- API Key 있으면 모든 기능 사용
- 클로닝된 목소리 저장
