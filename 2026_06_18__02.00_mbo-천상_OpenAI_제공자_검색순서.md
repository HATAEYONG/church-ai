# MBO 목표서 — OpenAI 제공자 추가 + 검색순서 설정

**승인 일시**: 2026-06-18 (PO 승인, 배포까지)

## 최상위 목표
"멘토·설교의 AI 제공자로 OpenAI를 추가하고, 설정의 검색순서(제공자 우선순위)대로 라우트가 제공자를 선택해 동작하게 한다. 키 0개면 목업."

## 설계
0. openai 패키지 추가
1. 서버 ai-providers.ts: 제공자 메타 + resolveProvider(req)(검색순서·키로 선택) + 통합 streamChat(Anthropic/OpenAI)
2. 클라 ai-config.ts: OpenAI 키 + 검색순서 저장, 헤더 x-anthropic-key·x-openai-key·x-ai-order
3. 설정 UI: OpenAI 키 필드 + 검색순서 재정렬(↑/↓) 카드
4. mentor·sermon 라우트가 resolveProvider로 제공자 결정, 없으면 목업
5. build·tsc·lint + 브라우저 검증 + 프로덕션 배포

- 검색순서 동작: 순서상 키가 설정된 첫 제공자 사용
- 기본 모델: Anthropic claude-opus-4-8, OpenAI gpt-4o (env OPENAI_MODEL 변경 가능)

## 완료·KPI
- 검색순서대로 제공자 진입(헤더·분기 검증; 실호출은 유효키 전제)
- 키 0개 목업 유지 / OpenAI 키만 있으면 OpenAI 경로
- build·tsc·lint 통과, 데스크톱·모바일 무회귀, 배포
