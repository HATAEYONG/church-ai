# 음성 기능 사용 가이드

이 가이드는 이 플랫폼의 AI 음성 기능을 사용하는 방법을 안내합니다.

## 🎙️ 음성 기능 소개

이 플랫폼은 일레븐랩스(ElevenLabs)의 AI 음성 합성 기술을 사용하여 텍스트를 자연스러운 목소리로 변환해줍니다.

### 주요 기능

- **기도/감사 노트 TTS**: 작성한 노트를 음성으로 재생
- **AI 멘토 음성 답변**: AI 멘토의 응답을 음성으로 들음
- **성경 읽기**: 성경 구절을 AI 목소리로 읽기
- **목소리 선택**: 다양한 목소리와 속도, 음조 조절

## 🚀 시작하기

### 1단계: ElevenLabs API Key 발급

1. **ElevenLabs 가입**
   - https://elevenlabs.io 접속
   - 회원가입 (무료 플랜 제공)

2. **API Key 발급**
   - 로그인 후 우측 상단 프로필 → **"Settings"**
   - **"API Keys"** 탭 클릭
   - **"Create New API Key"** 버튼 클릭
   - API Key 복사

3. **환경 변수 설정**
   ```env
   # .env.local 파일에 추가
   ELEVENLABS_API_KEY=your-elevenlabs-api-key-here
   ```

### 2단계: 개발 서버 재시작

```bash
# 현재 서버 중지 (Ctrl+C)
# 서버 재시작
npm run dev
```

## 📱 음성 기능 사용

### 음성 기능 데모 페이지 (`/voice`)

1. **접속**
   - 브라우저에서 `http://localhost:3000/voice` 접속

2. **샘플 텍스트 선택**
   - 기도, 감사, 성경, AI 멘토 샘플 중 선택
   - 또는 직접 텍스트 입력

3. **음성 재생**
   - **"음성 듣기"** 버튼 클릭
   - AI 목소리로 텍스트를 읽어줍니다

### 목소리 설정 페이지 (`/voice-settings`)

1. **접속**
   - `http://localhost:3000/voice-settings` 접속

2. **목소리 선택**
   - 6가지 목소리 중 선택 (Rachel, Adam, Josh, Liam, Emily, Daniel)
   - 각 목소리의 설명 확인

3. **음성 세부 설정**
   - **안정성 (Stability)**: 0~100% (낮을수록 표현력 풍부)
   - **유사성 부스트 (Similarity Boost)**: 0~100% (목소리 특성 유지)
   - **스피커 부스트 (Speaker Boost)**: 클라리티 강화

4. **테스트 및 저장**
   - 테스트 텍스트로 재생 확인
   - **"설정 저장"** 버튼으로 localStorage에 저장

## 🎯 활용 시나리오

### 시나리오 1: 기도 노트 음성 재생

```
1. /prayer 접속
2. 작성한 기도 노트 상세 보기
3. 🎙️ 음성 재생 버튼 클릭
4. 기도 내용을 AI 목소리로 듣기
```

### 시나리오 2: 감사 노트 음성 모음

```
1. /gratitude 접속
2. 일일 감사 노트 목록
3. 전체 음성 재생으로 오늘의 감사 들려주기
```

### 시나리오 3: AI 멘토와 대화

```
1. /mentor 접속
2. "요셉에 대해 알려줘" 입력
3. AI 멘토 응답 수신
4. 음성 재생 버튼으로 응답 듣기
```

### 시나리오 4: 성경 읽기

```
1. /games/quiz 접속 (성경 퀴즈)
2. 퀴즈에 포함된 성경 구절
3. 🎙️ 음성으로 성경 읽기
```

## ⚠️ 주의사항

### API 사용 한도

**무료 플랜 (Free Tier)**
- 월 10,000자 변환 가능
- 약 5-10분 분량의 음성
- 초과 시 유료 전환 필요

**사용량 확인**
- ElevenLabs Dashboard → **Usage**
- 현재 사용량 및 남은 한도 확인

### 네트워크 요구사항

- **인터넷 연결 필수**: TTS는 클라우드 API에서 처리
- **지연 시간**: 텍스트 길이에 따라 1-5초 소요
- **오프라인**: 지원하지 않음

### 브라우저 호환성

- **권장**: Chrome, Edge, Firefox (최신 버전)
- **Audio API 지원**: HTML5 Audio API 필요
- **Safari**: 일부 기능 제한 가능

## 🔧 문제 해결

### 문제: "음성 변환 실패" 메시지

**원인 1: API Key 미설정**
- `.env.local`에 `ELEVENLABS_API_KEY` 확인
- 값이 비어있거나 형식이 올바르지 않음

**원인 2: API 한도 초과**
- ElevenLabs Dashboard에서 사용량 확인
- 한도 초과 시 다음 달 대기 또는 유료 플랜 전환

**원인 3: 네트워크 오류**
- 인터넷 연결 확인
- 방화벽/프록시 설정 확인

### 문제: 음성 품질 저하

**해결 방법:**
- 목소리 설정에서 안정성 낮추기 (20-30%)
- 유사성 부스트 높이기 (80-90%)
- 스피커 부스트 활성화

### 문제: 재생 속도 너무 느림

**해결 방법:**
- 텍스트 길이 줄이기 (200자 권장)
- 네트워크 속도 확인
- 다른 목소리로 시도
