# 보이스 클로닝 가이드

이 가이드는 에이맨(A-Men)의 보이스 클로닝 기능을 사용하는 방법을 안내합니다.

## 🎤 보이스 클로닝이란?

보이스 클로닝은 사용자의 목소리를 AI로 복제하여 TTS(Text-to-Speech)에 사용하는 기능입니다. 내 목소리로 AI가 텍스트를 읽어줍니다.

### 주요 기능

- **목소리 생성**: 오디오 샘플로 내 목소리 생성
- **목소리 관리**: 생성한 목소리 조회/삭제
- **목소리 사용**: 클로닝된 목소리로 TTS 재생
- **단순화된 설정**: 남자1, 여자1 2개 기본 목소리

## 🚀 시작하기

### 1단계: ElevenLabs API Key 발급

보이스 클로닝은 ElevenLabs API Key가 필요합니다.

1. **ElevenLabs 가입**
   - https://elevenlabs.io 접속
   - 회원가입 (무료 플랜 가능)

2. **API Key 발급**
   - 프로필 → Settings → API Keys
   - "Create New API Key" 클릭
   - 키 복사

3. **API Key 설정**
   ```env
   # .env.local 또는 Vercel 환경 변수
   ELEVENLABS_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
   ```

### 2단계: 샘플링 오디오 준비

**샘플링 요구사항:**
- **길이**: 최소 10초 이상 권장
- **품질**: 잡음이 없는 깨끗한 오디오
- **발음**: 다양한 톤과 감정 표현
- **형식**: MP3, WAV, M4A 등 오디오 파일

**샘플링 가이드:**
```
1. 평소 목소리로 읽기 (대화하듯 자연스럽게)
2. 다양한 톤 (높고 낮음)
3. 감정 표현 (기쁨, 웃음 등)
4. 1-3분 분량 권장 (전체 10초 이상)
```

**녹음 예시:**
```
❌ 잡음 많은 환경 (카페, 거리)
❌ 배경음악 있는 오디오
❌ 전화음질, 화상 통화
✅ 조용한 스튜디오
✅ 조용한 환경에서 녹음
✅ 고품질 마이크로 녹음
```

## 📱 보이스 클로닝 사용법

### 목소리 생성 단계

#### 방법 A: 파일 업로드

1. **/voice-clone 접속**
   ```
   https://church-ai-rho.vercel.app/voice-clone
   ```

2. **목소리 이름 입력**
   - 예: "나의 목소리", "목소리1" 등

3. **설명 입력 (선택사항)**
   - 목소리 특징 설명

4. **오디오 파일 업로드**
   - "파일 선택" 클릭 또는 드래그 앤 드롭
   - 1개 이상 오디오 파일 선택
   - 최소 10초 이상 권장

5. **목소리 생성 시작**
   - "목소리 생성 시작" 버튼 클릭
   - 5-10분 소요 (기다려 주세요!)

6. **생성 완료**
   - 성공 메시지 표시
   - 목소리 설정 페이지에서 사용 가능

#### 방법 B: 마이크로 녹음 (추후 개발)

- 향후 마이크로 녹음 기능 추가 예정
- 현재는 파일 업로드만 지원

### 목소리 관리

**생성된 목소리 목록:**
```
/voice-clone 페이지 하단에 표시
- 목소리 이름, 설명, 샘플 수, 생성일
- 삭제 버튼으로 제거 가능
```

**목소리 삭제:**
```
1. 목소리 오른쪽 "삭제" 버튼 클릭
2. 확인 다이얼로그에서 "확인" 클릭
3. 목소리 삭제 완료
```

## 🔧 문제 해결

### 문제 1: 목소리 생성 실패

**에러 메시지:**
```
"목소리 클로닝 실패"
```

**원인 및 해결:**
- **원인 1**: API 키 미설정
  - 해결: .env.local에 ELEVENLABS_API_KEY 입력

- **원인 2**: 샘플링 품질 낮음
  - 해결: 10초 이상 깨끗한 오디오 사용

- **원인 3**: 파일 형식 잘못됨
  - 해결: MP3, WAV, M4A 형식 사용

- **원인 4**: 한도 초과 (최대 3개)
  - 해결: 기존 목소리 삭제 후 재생성

### 문제 2: 생성된 목소리 품질 저하

**해결 방법:**
```
1. 더 긴 오디오 샘플 제공 (1-3분)
2. 다양한 톤과 감정 표현
3. 잡음 없는 환경에서 재녹음
4. 전문적인 녹음 장비 사용
```

### 문제 3: 목소리 목록에 안 보임

**해결 방법:**
```
1. 페이지 새로고침
2. localStorage 확인
3. ElevenLabs Dashboard에서 Voices 확인
```
